# 👥 Otimização de Escalas/Turnos

> Otimização de escalas e turnos com IA. Equilibra demanda, preferências e legislação para escalas justas que reduzem custo e horas extras.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/otimizacao-escalas/

[![Vertical](https://img.shields.io/badge/Vertical-RH-blue)]() [![IA](https://img.shields.io/badge/IA-Optimization-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**-34%** Horas extras · **98%** Cobertura · **+22%** Satisfação · **-17%** Custo

## 🧠 Tecnologias de IA
Optimization · ML · Constraint Programming

## ✨ O que faz
- Escala otimizada (restrições CLT)
- Balanceamento de demanda
- Preferências do colaborador
- Redução de horas extras

## 🗄️ Fonte de dados real
[Demanda + CLT](https://www.gov.br/trabalho)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Demanda + CLT
2. **Processamento com IA** — Optimization · ML · Constraint Programming
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
